Natus Neuroworks provides continuous EEG (CEEG) monitoring which help to detect neurologic deterioration in patients with brain insults such as traumatic brain injury, stroke, and status epileptics.Natus also allows for the CEEG recordings to be available rapidly for interpretation by skilled readers from remote locations (e.g. EEG lab, office, home).

There are nine mobile EEG carts which a vendor PC running Windows XP, which monitoring hardware attached that function primarily in the BIDMC West ICUs (mainly trauma and surgical ICUs and neurological ICU, but also CCU and medical ICUs). EEG is based out of Baker 5 where there are 5 reading stations with the software loaded as well.

This is an FDA-regulated solution that will store clinical ePHI data. This solution will function 24x7 with a maximum downtime tolerance of 2 hours. If the testing equipment is unable to contact the server the data will be stored on the workstations until the server is available again.

Persyst is a related application that lives on the application server that allows the manipulation and viewing of the EEG images.

FAQs

    Q: After an EEG Monitor cart is moved to a new location it loses its connectivity to the Natus system

    A: When Natus Monitor machines are moved to a different location a new DHCP IP address needs to be assigned to the machine based on that new location. In order for this to happen, the machine needs to release it's previous IP address to acquire the new one. This happens normally when a machine is shut down properly. When it is not shut down properly and the original IP address is retained in the DDNS tables so follow these tasks after the cart is in the new location:

    1. Click the Start menu button on the Windows taskbar

    2. Click Run... on this menu.

    3. If the Natus workstation is holding a current IP address, type 'cmd' (without the quotes) in the text box that appears. A command prompt window appears on the screen.

    4. Type 'ipconfig /release' to let go of the address and press Enter

    5. Then type 'ipconfig /renew' and press Enter to obtain the new IP address

    6. Close out of the black window and you are all set
